# `sdks` folder

I cannot upload the files from the sdks folder to GitHub due to
licence restructions, but you can obtain these files elsewhere.

## VST SDKs

You only need some folders from VST SDKs, the majority of which you can find here:
 - https://github.com/steinbergmedia/vst3sdk

There might be a problem with obtaining files for `sdks/pluginterfaces/vst2.x`.
Please refer to the file list below, and by googling the file names you will
find places where they could be downloaded.

## AAX SDK
AAX might be tricky, as it requires signing an agreement with Avid, more
information is available at this page. It's a manually handled process,
and you won't be able to get an SDK without talking to their people.
  - https://www.avid.com/alliance-partner-program/how-to-apply

For development you might want to edit the CMake project and build scripts
to simply switch off AAX building.
